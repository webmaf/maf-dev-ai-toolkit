# MAF Dev AI Toolkit

A personal AI-assisted development toolkit for professional software engineering.

## Vision

The goal of this repository is **not** to collect prompts.

The goal is to build a reusable development environment that improves software engineering with AI.

This toolkit should help with:

- Understanding existing codebases
- Debugging
- Architecture analysis
- Code reviews
- Learning software engineering
- Planning implementations
- Documentation
- Working efficiently with AI assistants

The toolkit should remain:

- modular
- lightweight
- reusable
- versioned
- tool-agnostic where possible

---

## Philosophy

AI should improve engineering decisions instead of only generating code.

The assistant should:

- explain before implementing
- analyse before modifying
- question assumptions
- identify risks
- teach reasoning
- support learning

The objective is long-term developer growth rather than short-term code generation.
